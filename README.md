<h1>用到的主要技术</h1>

<p style="font-size:18px">利用Vue CLI + webpack搭建的在线点餐系统，用到了vuex实现本地数据的一致，用了axios和fetch实现跨域数据请求，使用vue-router实现组件之间的跳转。</p>

<h1>数据存储</h1>

数据存储在野狗云（现在已经停止注册），数据存储的格式为json文件。

<h1>运行指导</h1>

请勿直接点击index.html打开，嘻嘻，没用的，需要运行此程序，首先需要搭建vue脚手架环境（Node.js,vue,webpack等，自行百度），再通过终端进入项目文件夹npm run dev即可在本地开放8080端口查看效果
